# Portfolio Website

A simple personal portfolio website for Sachithanantham P.

## Files
- `index.html` — portfolio landing page
- `styles.css` — site styling
- `resume.html` — professional resume page
- `resume.md` — resume content in Markdown format
- `1782912008557.png` — profile photo
- `.github/workflows/pages.yml` — GitHub Pages deployment workflow

## Run locally

Open `index.html` in a browser, or serve the folder with a local server:

```bash
python -m http.server 8000
```

Then visit http://localhost:8000
